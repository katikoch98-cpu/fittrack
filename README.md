# ⚡ FitTrack

Persönliche Fitness-Tracking App – optimiert für iPhone als PWA.

## Features
- 🏋️ Gym-Einheiten mit Übungen, Sets, Wiederholungen & Gewicht
- 🏃 Jogging-Sessions mit Distanz, Zeit & Pace
- 📅 Kalender mit detaillierter Workout-Planung
- 🔁 Wiederkehrende Workouts (wöchentlich / 2-wöchentlich)
- 📱 PWA – als App auf dem iPhone Homescreen installierbar

## Deployment auf Vercel (kostenlos)

1. Dieses Repository auf GitHub hochladen
2. Auf [vercel.com](https://vercel.com) einloggen (kostenlos mit GitHub)
3. „Add New Project" → dieses Repository auswählen
4. Framework: **Vite** wird automatisch erkannt
5. Auf **Deploy** klicken → fertig!

## Lokale Entwicklung

```bash
npm install
npm run dev
```

## Auf iPhone installieren

1. Die Vercel-URL in **Safari** öffnen
2. Teilen-Symbol → „Zum Home-Bildschirm"
3. App heißt dann „FitTrack" und öffnet sich ohne Safari-Leiste
